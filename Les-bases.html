Les formulaires

  • La balise Form permet de créer un Formulaire
  • name :Permet de donner un nom au formulaire
  • action :Cet attribut est OBLIGATOIRE et permet de renseigner le script, la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du formulaire.Cette page va reçevoir l'ensemble des données saisie par mon utilisateur
  • method :Permet de préciser quellle méthode [GET | POST] doit être utilisée par le navigateur pour transmettre les données saisies vers la page action















Créer un lien en HTML :

  • La balise "a" permet de créer un lien vers une autre ressource. (Images, Vidéo, PDF, ZIP...)
  • href: Permet de définir le chemin vers le fichierà lier
  • target: Prend plusieurs valeurs : _blank pour ouvrir le lien dans une nouvelle fenêtre par exemple
  • title: Permet de donner un titre au lien. Ce titre afficher dans une petite bulle au survol du lien par la souris
